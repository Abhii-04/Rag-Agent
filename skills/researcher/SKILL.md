# Research Skill

## Purpose

Use this skill whenever the task requires gathering information from external sources.

## Procedure

1. Understand the user's request.
2. Decide which search tool is appropriate.
3. Search multiple sources when necessary.
4. Compare results.
5. Remove duplicated or outdated information.
6. Produce a concise summary.
7. Include important facts, dates, and references.
8. If information is uncertain, explicitly mention it.

## Best Practices

- Prefer authoritative sources.
- Search again if results conflict.
- Do not rely on a single source for important claims.
- Keep summaries factual.
- Avoid speculation.

## Output Format

### Summary

A concise explanation.

### Key Findings

- Finding 1
- Finding 2
- Finding 3

### Sources Used

- Source 1
- Source 2

### Confidence

High / Medium / Low
