# Web Search Skill

## Purpose

Use this skill when a task requires current, recent, or source-backed web information.

## Procedure

1. Turn the user's request into focused search queries.
2. Prefer authoritative and primary sources.
3. Use multiple searches when the answer depends on recency or conflicting reports.
4. Compare source dates and facts.
5. Summarize only what the sources support.
6. Include source names or URLs in the final answer when the user needs evidence.

## Best Practices

- Treat latest, current, recent, today, and similar wording as requiring web search.
- Do not fabricate citations.
- Make uncertainty explicit when sources disagree.
- Use Wikipedia only for background, not for breaking or current facts.
